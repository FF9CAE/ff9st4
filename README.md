# 📒 FF9's Sublime Text 4 Configs
***FF9's Sublime Text 4 Configs*** is a prebuilt ***Preferences.sublime-settings*** you can easily tune to your liking.

## ✅ Features
- Easy to change as it has comments (`e.g. //`) to guide you.
- Readable font size
- ###### ...and some small updates to the UI you will love for the speed and aesthetics.

## ⬆ Installation
1. Either use:
- `git clone https://github.com/FF9CAE/ff9st4.git` to clone the config.
- OR `gh clone FF9CAE/ff9st4` to clone the config using GitHub CLI.
- OR `Releases > ff9st4 1.1` to download the config's latest files.

2. Open the `ff9st4` folder, and open the `.preferences` subfolder.

3. Now, copy the `Preferences.sublime-settings` to `C:/Users/(user)/AppData/Roaming/Sublime Text/Packages/User`, this is where the default `Preferences.sublime-settings` lives. Replace it with the one you downloaded from this repo.

4. Now you have a custom starting file for all the Sublime Settings.
